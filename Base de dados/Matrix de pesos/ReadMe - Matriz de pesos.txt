Conexoes: link entre regiões
Matrix de conexões: matrix que tem informacao do link entre as regioes considerando origem e destrino. Na matrix de conexoes a diagonal (regiao i contra regiao i) é zero.

Matrix de pesos: matrix que tem informacao do link entre as regioes, padronizando que a soma das colunas seja igual a um.

MatrixW-v2 : Calculo das conexoes entre amcs.
Wmatrix_v3 : Calculo da matrix de conexoes entre amcs, completa.
Wmatrix_v4 : Calculo da matrix de pesos entre amcs, completa.
